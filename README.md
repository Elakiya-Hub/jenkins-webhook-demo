# Jenkins GitHub Webhook Pipeline Demo

1. Create a local Git repository.
2. Push the project to GitHub.
3. Configure a Jenkins Pipeline project.
4. Connect Jenkins with the GitHub repository.
5. Configure GitHub Webhook.
6. Push code changes to GitHub.
7. Jenkins automatically detects the push.
8. Jenkins executes the `demo.sh` script.
9. Jenkins sends an email notification after the build.

## Output

- Automatic Jenkins build after every GitHub push.
- Successful execution of `demo.sh`.
- Email notification sent after build completion.
